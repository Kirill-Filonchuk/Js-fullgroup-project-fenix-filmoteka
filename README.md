# Командный проект FILMOTEKA

Цели проекта:

1. Цели проекта:

1) в заданные сроки командно реализовать веб-приложение. Срок выполнения проекта для групп Online -
   14 дней (например, понедельник вечером начало проекта - через понедельник вечером презентация
   проекта).

2) Получение практики работы в командах:

- опыт работы со сборщиком проектов Parcel;
- командное взаимодействие и работа c Git; (Видео по работе с Git:
  https://drive.google.com/file/d/1TMV7dICcE1wPn-afdjfcVoffSH9wGt 5K/view)
- навык ветвления и слияния проекта;
- решение конфликтов после слияния веток;
- опыт планирования и постановки задач;
- опыт работы с Trello (https://trello.com/);
- опыт на позиции team lead команды;
- опыт на позиции scrum master команды;
- опыт презентации проектов.

3. Выполненные задачи

-01 Сверстать хедер главной страницы (мобилка, планшет, десктоп) -02 По нажатию на логотип/кнопку
"Home" отрисовывается главная страница -03 По нажатию на кнопку "My Library" отрисовывается
библиотека пользователя -04 Вёрстка футера (мобилка, планшет, десктоп) (одинаковый для двух страниц)
-05 Сверстать шаблон карточки одного фильма -06 Сверстать контейнер главной страницы для правильного
позиционирования карточек фильмов (мобилка, планшет, десктоп) -07 Реализовать подгрузку популярных
фильмов на главную (первую) страницу -08 Сверстать или подключить номера страниц (пагинация) на
главной странице -09 При переходе на каждую страницу отрисосывать соответствующую часть фильмов -10
Реализовать поиск и отрисовку фильмов по ключевому слову -11 Сверстать контейнер страницы "My
Library" для правильного позиционирования карточек фильмов (мобилка, планшет, десктоп) -12 Сверстать
хедер для страницы "My Library" (мобилка, планшет, десктоп) -13 По нажатию на логотип отрисовывается
главная страница -14 По нажатию на кнопку "Watched" показываются просмотренные фильмы пользователя  
-15 По нажатию на кнопку "Queue" показываются фильмы добавленные в очередь пользователя -16
Сверстать шаблон модалки одного фильма (мобилка, планшет, десктоп) -17 По нажатию на карточку фильма
на любой странице должна открываться модалка с динамически подставленной информацией о фильме -18 По
нажатию на кнопку "Add to watched" фильм добавляется в просмотренные фильмы текущего пользователя
(local-storage) -19 По нажатию на кнопку "Add to queue" фильм добавляется в очередь текущего
пользователя (local-storage) -20 Реализовать закрытие модалки по нажатию на клавишу ESC и по клику
вне области модалки, не забыть снять слушатели -21 Подключить/сверстать Loader (spinner) для
асинхронных запросов -22 Сверстать модалку (мобилка, планшет, десктоп) с информацией о участниках
вашей команды, которая будет открываться по нажатию на текст "GoIT Students" в футере
